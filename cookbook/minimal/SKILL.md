# Minimal Recipe — F0WCUS Lightweight

**For users who want basic focus sessions with no customization.**

## What's Included
- `BEGIN` / `END` sessions
- Basic daily handoffs
- Simple open loops (no priority tiers)

## What's Excluded
- No priority levels (HIGH/MED/BACKLOG)
- No session templates (quick/standard only)
- No setup wizard (defaults only)
- No `#decision` promotion
- No GitHub backup
- No 1% Better, Witness Protocol, Emotion Check
